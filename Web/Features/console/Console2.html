Command Line Tool

As an alternative to this website, you can download this console application that will scan your source and output textual and graph data.

file_download Download

Usage

Specify the output directory and one or more directories to scan. The application will recursively scan the speciifed directories looking for packages.config files.

CanIHazDotnetCore.exe <output_directory> <dir_to_scan_1> [dir_to_scan_2] ... [dir_to_scan_n]

For example, to scan your whole source tree

CanIHazDotnetCore.exe c:\temp\output c:\source\MyApplicaion

or only scan specific projects: (Current directory being the source root)

CanIHazDotnetCore.exe c:\temp\output MyApplication.Data MyApplication.Web Tests\Tests Tests\Tests.UI

Output

Each found packages.config file is tagged with the name of the directory it was found in as the 'Project' name. The application will output:
  • Flat.txt - A flat list of the dependencies
  • Tree.txt - A tree of the dependencies
  • All.gv - A GraphVis file of all dependencies of all projects
  • 1Level.gv - A GraphVis file of direct dependencies of all projects
  • <Project>.gv - A GraphVis file of dependencies for <Project>

Result Statuses and Colours

The textual output will use a textual status (eg [Supported]), while the GraphVis data uses colour. The possible statuses are:
Your input projects.config files
The latest release version of the package supports .NET Standard
The latest release version of the package does not support .NET Standard, but the latest pre-release version does
We have identified that the packages as no longer being required on .NET Standard, or there is a a replacement package with a different name
The latest release and pre-release versions of the package do not support .NET Standard
This package could not be found on nuget.org, it is most likely from an internal nuget feed
Something went wrong when inspecting the package

GraphVis

The .gv file is a GraphVis dot file. It can be used with the GraphVis application, online at WebGraphViz or with many other web and non-web based network graphing applications.
